# Sample Life Maps

This directory contains example life map files for demonstration purposes.

## Files

- **mikes-life-map.lifemap** - A comprehensive example showing various life events, relationships, career progression, and personal goals

## Important Note

**All sample data is completely fictional and created for demonstration purposes only.** 

The names, dates, places, and events in these files do not represent real people or actual events. They are designed to showcase the features and capabilities of the Life Map application.

## Using Sample Files

1. Open Life Map
2. Go to File → Open
3. Navigate to the `samples/` directory
4. Select a `.lifemap` file to open

## Creating Your Own

Feel free to use these samples as templates for your own life map:

1. Open a sample file
2. Go to File → Save As
3. Save with a new name
4. Edit the content to reflect your own life

## Privacy

When creating your own life maps:
- All data is stored locally on your computer
- No information is sent to external servers
- `.lifemap` files are plain JSON - you can inspect them in any text editor
- Be mindful when sharing files as they may contain personal information
